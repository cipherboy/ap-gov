Kryptos by J. Sanborn commissioned in 1989.

Background

  • The CIA was building a new headquarters in Langley, VA
  • They wanted art that is “indicative of the CIA’s work”
  • Spent $250,000 on a commission

James Sanborn, Artist

  • Born in Washington, D.C.
  • Known for using natural materials that create a sense of mystery
  • Got commission after design was reviewed by NEA and CIA
  • Worked four months with a retired CIA cryptographer and prominent fiction writer

Kryptos

  • Contains three parts
    • Petrified tree support
    • Bubbling pool
    • Copperplate screen
  • Screen contains 1735 letters
    • Contains the all important code
    • A great challenge for CIA and NSA employees

Breaking the Code

  • In 1998, a physicist for CIA announced that he has cracked code using paper and pencil
  • A year later a computer scientist cracked it using computers
  • The last section of 97 characters, called K4, remains uncracked
